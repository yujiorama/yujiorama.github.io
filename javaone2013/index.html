Reveal.js とは

私も今日(2012/01/11)の朝にコリスさんの記事で知ったばかりなのでそんなに詳しくないのですがざっくり言うと,

CSS3 のアニメーションを使って簡単にオシャレなプレゼン用スライドショーを作ることができるライブラリ

です. 中のコードを見る限りすんごい良くできています. JavaScript と CSS3 をうまく連携させて作ってあるので, 3D による演出もスムーズですし, Transition や Theme の追加も簡単にできちゃいます.

Step 00 - reveal 用スタイルシート読み込み

必要なスタイルシートを読み込みます. 


<link href="http://fonts.googleapis.com/css?family=Lato:400,700,400italic,700italic" rel="stylesheet" type="text/css">
<link rel="stylesheet" href="css/reset.css">
<link rel="stylesheet" href="css/main.css">
<link rel="stylesheet" href="lib/zenburn.css">

Step 01 - reveal 用要素とスライダを設置

id に reveal を持つ要素を設置しその中に, class に slides を持つ要素を設置してください. そのなかにスライドの内容を書きこんでいきます. 


<body>
    <div id="reveal">
        <div class="slides">
            <!-- スライド 00 -->
            <section>
                <h1>Title</h1>
                <h3 class="inverted">sub title</h3>
            </section>
            <!-- スライド 01 -->
            <section>
                <h2>page 01</h2>
                <p>Text Text Text Text</p>
            </section>
            <!-- スライド 02 -->
            <section>
                <h2>page 02</h2>
                <p>Text Text Text Text</p>
            </section>
        </div>
    </div>
</body>

Step 02 - コントローラ用要素を設置

画面右下に表示するコントローラを設置します. id に reveal を持つ要素の中に設置してください. 表示する文字は自由に変えられます. 


<aside class="controls">
    <a class="left" href="#">◄</a>
    <a class="right" href="#">►</a>
    <a class="up" href="#">▲</a>
    <a class="down" href="#">▼</a>
</aside>

Step 03 - 進行バー用要素を設置

画面下に表示する進行バー用要素を設置します. id に reveal を持つ要素の中に設置してください. 


<div class="progress"><span></span></div>

Step 04 - reveal 用スクリプト読み込み

reveal 用スクリプトを読み込みます. 


<script src="js/reveal.js"></script>
<script src="lib/highlight.js"></script>

Step 05 - reveal を初期化 & パラメータ設定


<script>
    Reveal.initialize();
</script>

reveal を初期化します. 下記のコードでは引数に何も渡していませんが, 色々なパラメータに対応しています. 詳しくはこちらのスライドを御覧ください.

Reveal.js のパラメータ

  • controls … コントローラの表示/非表示フラグ
  • progress … 進行バーの表示/非表示フラグ
  • history … 履歴を url で記憶するかのフラグ
  • transition … トランジションの種類
  • theme … テーマの種類
  • rollingLinks … リンクをルーリング演出するかのフラグ

controls

画面右下にある十字キーみたいなやつの表示/非表示を切り替えることができます. デフォルト値は false です.

true / false

progress

画面下にあるページ進行状態を表すバーの表示/非表示を切り替えることができます. デフォルト値は false です.

true / false

history

ハッシュタグ(url の #~)でページ位置を記憶するかどうかのフラグです. デフォルト値は false です.

true / false

transition

transition の設定. 設定に応じてデザインやページ切り替え時の演出が変わります.

  • Default
  • Concave
  • Linear
  • Cube
  • Page

theme

テーマ切り替え. 現状はリンクの色が変わるぐらい?です. 今後色々増えていくんじゃないかなと思います.

  • Default
  • Neon

rollingLinks

リンク(aタグ)にマウスを乗っけた時に 3D 演出をするかどうかのフラグです. デフォルト値は true です.

true / false

Reveal.js をオーバービュー

SPACEキーを押すとオーバービューモードになります.

もう一度押すと元に戻ります. 全体を見渡せて便利です!!

ちなみに初めからオーバービューモードにしたいときは


<div id="reveal"> ... </div>
↓
<div id="reveal" class="overview"> ... </div>

とすれば OK!!
